## Mendelian Randomization Plots

### `mr_forest_bidirectional.png`
- **Type**: Forest plot
- **Content**: Bidirectional MR results comparing effects in both directions
- **Left panel**: Ratio → Size (β=2.0956, P=2.96e-126)
- **Right panel**: Size → Ratio (β=0.3788, P=3.10e-133)
- **Visual feature**: Asymmetry clearly visible with different effect sizes

### `mr_scatter_ratio_to_size.png`
- **Type**: Scatter plot
- **Direction**: Oil:Protein ratio → 100SW
- **X-axis**: SNP effects on ratio (exposure)
- **Y-axis**: SNP effects on size (outcome)
- **Regression line**: Slope = 2.0956 (IVW estimate)
- **Error bars**: Standard errors for each SNP

### `mr_scatter_size_to_ratio.png`
- **Type**: Scatter plot
- **Direction**: 100SW → Oil:Protein ratio
- **X-axis**: SNP effects on size (exposure)
- **Y-axis**: SNP effects on ratio (outcome)
- **Regression line**: Slope = 0.3788 (IVW estimate)
- **Error bars**: Standard errors for each SNP

## Genetic Architecture Plots

### `genetic_correlation_matrix.png`
- **Type**: Heatmap
- **Content**: Genetic correlation matrix between all traits
- **Traits**: 100SW, Protein, Oil, Oil:Protein ratio
- **Diagonal**: Self-correlations (1.0)
- **Key feature**: High correlation (0.334) between ratio and size

### `conditional_r2_plot.png`
- **Type**: Bar/Scatter plot
- **Content**: Marginal vs. conditional R² values
- **Left bars**: Marginal R² (ratio & size: 0.334)
- **Right bars**: Conditional R² (both < 1e-30)
- **Visual message**: Near-zero residual correlation after conditioning

## Visualization Specifications

### Color Scheme
- **GWAS plots**: Chromosomes in alternating colors
- **MR plots**: Exposure in blue, outcome in red
- **Correlation plots**: Gradient from blue (negative) to red (positive)
- **Comparison plots**: Consistent palette across panels

### Dimensions
- **Standard size**: 1200×800 pixels
- **Multi-panel**: 1600×1200 pixels
- **Resolution**: 300 DPI for publication
- **Format**: PNG with transparent background

### Annotation
- **All plots**: Title, axis labels, legend
- **Significance**: Threshold lines and labels
- **Statistics**: Key values (P, β, R²) annotated on plot
- **Credits**: Pipeline version and date
